Y.M Apps Press Kit
====================

このZIPには、Y.M Apps（PINO が個人開発しているプロダクト群）の
レビュー・媒体掲載・SNS紹介などにご利用いただける素材が含まれます。

== 含まれるもの ==
- pinokun/                 マスコット「ぴのくん」のアイコン・主要スタンプ
- apps/                    各アプリのカバー・スクリーンショット
- info.md                  各プロダクトの1行紹介とリンク

== 出典の表記について ==

記事内にサイトのURLを掲載いただけると幸いです。
